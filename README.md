# Trivia-Quiz

This is a JavaScript Trivia website design to provide users with an interactive and engaging trivia experience. The website allows users to test their knowledge on JavaScript by answering multiple choice questions regarding it.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Credits](#credits)

## Installation

To install and run the Trivia website locally, follow these steps:
1. Clone the repository
2. Navigate to the Trivia-Quiz directory
3. Open the Trivia-Quiz.html file in your browser

or 

1. Click on the link in the about section of the repository.

## Usage

The Trivia-Quiz is a trivia website that allows users to test their knowledge on JavaScript. Users can answer questions in the form of multiple choice questions. Users can select the correct answer to advance to the next question.
